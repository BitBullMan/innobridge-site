Review-First Automation

The system is built for quality control before account automation.

Supabase queue

`social_content_queue` stores each generated pack with status, priority, language mode, source summary, review notes, and approval timestamps.

n8n workflow

Daily trigger collects intelligence inputs, calls AI, saves the pack, and emails the operator for review before publishing.

Performance feedback

`social_content_feedback` captures impressions, likes, comments, saves, clicks, subscriptions, and notes by platform.
